The User Interface.

The main interface consists of a menubar, a toolbar, the currently viewed page, and the status bar.

the menu

The menubar consists of 3 pulldown-menu's, called "File", "Settings", and "Help". The following options are available in each of these menu's.

  • File

  • Settings

  • Channels

  • Help
